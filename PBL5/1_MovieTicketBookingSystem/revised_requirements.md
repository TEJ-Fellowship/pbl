Core Functionality (Must Implement)

1. Movie Management

   Add new movies to the system

   Update movie details (title, description, duration, genre, release date)

   Remove movies from the system

   Search and filter movies by various criteria (title, genre, release date, language)

2. Showtime Management

   Create showtimes for movies

   Update showtime details

   Delete showtimes

   View all showtimes for a specific movie or theater

3. Theater and Screen Management

   Add theater information

   Manage screens within theaters

   Configure seating arrangements for each screen

   Define seat categories (regular, premium, VIP) with different pricing

4. Booking System

   Browse available movies and showtimes

   Select seats for a show

   Reserve seats temporarily during booking process

   Confirm booking and generate tickets

   Cancel bookings (with applicable rules)

5. Payment Integration

   Integrate with payment gateway (simulate if needed)

   Process payments for bookings

   Handle payment success/failure scenarios

   Generate payment receipts
